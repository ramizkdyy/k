## Download Instructions:
1. Go to https://fontawesome.com/icons
2. Search for each icon name
3. Download as SVG format
4. Place in assets/icons/ folder with naming convention: icon-name-style.svg
   Example: arrow-left-solid.svg, home-regular.svg, user-light.svg
